## YFM(YAML Front Matter)

- markdown 파일의 최상단에 위치하며 3개의 하이픈으로 시작과 끝을 표시한다.
- YFM은 YAML을 사용해서 글의 제목, 날짜, 카테로기, 태그, 레이아웃 등을 정의할 수 있다.
